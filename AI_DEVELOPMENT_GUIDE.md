# 🚀 AI Development Guide - Urano AI News Template

Este repositorio está optimizado para trabajar con **Cursor, Claude, Windsurf, Continue.dev**, etc.

## Cómo empezar

1. Clona el repo:
   ```bash
   git clone https://github.com/tu-usuario/urano-ai-news-template.git
   cd urano-ai-news-template
   npm install
2. Abre la carpeta en Cursor (recomendado) o tu editor IA favorito.
3. Los archivos en .cursor/rules/ se cargarán automáticamente.
